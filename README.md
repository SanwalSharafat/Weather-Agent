Weather Agent with Long-Term Memory

A smart weather agent that provides real-time weather updates while storing user information for personalized experiences. User details and preferences are stored in a JSON-based long-term memory, allowing the agent to remember past interactions and improve recommendations over time.

Features

🌤 Real-Time Weather: Fetches current temperature, humidity, and weather conditions for any city.

💾 Long-Term Memory: Stores user preferences, frequently checked locations, and other info in JSON.
